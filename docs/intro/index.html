インストール手順

方法1: VCC (VRChat Creator Companion) 推奨

1

VCCを起動

VRChat Creator Companionを起動し、プロジェクトを選択します。

💡 画像: VCC起動画面とプロジェクト選択画面

2

リポジトリを追加

Settings > Packages > Add Repository で以下のURLを追加:

https://vpm.sakurayuki.dev/vpm.json

💡 画像: VCCのAdd Repository画面とURL入力

3

パッケージをインストール

Manage Project > Packages から「Unity Editor Asset Browser」を見つけて「+」ボタンでインストール。

💡 画像: VCCパッケージ一覧でのUEAB選択とインストール

方法2: Unity Package Manager

1

Package Managerを開く

Unity Editor で Window > Package Manager を選択。

💡 画像: UnityのWindow メニューからPackage Manager選択

2

Git URLでインストール

「+」ボタン > Add package from git URL を選択し、以下のURLを入力:

https://github.com/yuki-2525/UniAsset-UnityEditorAssetBrowser-.git

💡 画像: Package ManagerのGit URL入力画面

データベースセットアップ

Avatar Explorer データベース

1

Avatar Explorerをダウンロード

GitHubから最新版をダウンロードしてセットアップ。

💡 画像: Avatar Explorer起動画面とデータダウンロード状況

2

データベースパスを確認

Avatar Explorerインストールフォルダの Datas フォルダのパスをメモ。

例: C:\VRC-Avatar-Explorer\Datas

KonoAsset データベース

1

KonoAssetをダウンロード

GitHubから最新版をダウンロードしてセットアップ。

2

データベースパスを確認

KonoAssetのデータフォルダのパスをメモ。

例: C:\KonoAssetData

初回セットアップ

1

ツールを起動

Unity Editor で Window > Unity Editor Asset Browser を選択。

💡 画像: UnityのWindowメニューでのUEAB選択

2

設定画面を開く

右上の「設定」ボタンをクリック。

💡 画像: UEABウィンドウの設定ボタン位置

3

データベースパスを設定

先ほどメモしたパスを「AE Database Path」と「KA Database Path」に入力。

💡 画像: 設定ウィンドウでのデータベースパス入力

4

データベースを更新

「データベースを更新」ボタンをクリックしてデータを読み込み。

💡 画像: データベース更新ボタンと読み込み状況

トラブルシューティング

⚠️ データが表示されない

  • データベースパスが正しく設定されているか確認
  • Avatar Explorer/KonoAssetが正常に動作しているか確認
  • 「データベースを更新」ボタンを再実行

🐛 パッケージが見つからない

  • VCCのリポジトリURLが正しく追加されているか確認
  • Unityのバージョンが2022.3 LTS以上か確認
  • インターネット接続を確認

🖼️ 画像が表示されない

  • 画像キャッシュをクリア（設定から）
  • データベースの画像ファイルが存在するか確認
  • Unityを再起動

次のステップ

インストールが完了したら、実際の使い方を学びましょう！
