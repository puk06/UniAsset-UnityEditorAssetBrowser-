Avatar ExplorerとKonoAssetを統合したUnityエディタ上のアセット管理ツール

UnityエディタでVRChatアバターとアセットを効率的に管理・検索・インポートできるツールです。

※ Unity Editor Asset Browser のメインインターフェース

主な機能

🔍 統合検索

Avatar ExplorerとKonoAssetの両方のデータベースを横断検索できます。

📁 カテゴリ別表示

アバター、アバター関連アセット、ワールドアセット、その他に分類して表示。

🎯 詳細フィルタ

タイトル、作者、カテゴリ、対応アバター、タグ、メモの内容で詳細な検索が可能。

📦 ワンクリックインポート

Unitypackageの直接インポートとフォルダサムネイル自動設定。
